# US Dividend Screener
{: .no_toc }

Screen US stocks for high-quality dividend income opportunities. Uses TradingView Screener (no API key required) to filter for yield, valuation, and trend health. Replaces the US-only value-dividend-screener and dividend-growth-pullback-screener (which require paid FMP API) for US-market traders.
